# 📖 웹소설 집필 에이전트 팀 — 사용 가이드

> **대상:** 소설 창작 에이전트 팀을 활용하여 웹소설을 기획·집필하려는 창작자  
> **최종 수정:** 2026-05-10  
> **출처:** novel-team-agent v1 USAGE.md에서 이관, PGE 구조에 맞게 수정

---

## 1. 빠른 시작

> **"뭘 쓸지 정했다"** → Planner → Generator → Evaluator 전체 파이프라인  
> **"뭘 쓸지 고민 중"** → Planner(브레인스토밍 모드)  
> **"고민하면서 동시에 써야 해"** → 혼합 모드

---

## 2. 입력 데이터 준비 가이드

### 회차 집필 시 필요한 것

```markdown
## 집필 요청서

### 기본 정보
- 작품명: {제목}
- 장르: {장르 태그}
- 회차: 제{N}화
- 부제: {이번 화 부제 또는 키워드}

### 이전 화 연결
- 전화 마지막 상황: {1~2문장}
- 지속 중인 복선: {있다면}

### 이번 화 내용
- 핵심 사건: {무슨 일이 벌어지는가}
- 등장 인물: {누가 등장하는가}
- 감정 기조: {이번 화의 전체적인 분위기}
- 클리프행어 방향: {다음 화로 어떻게 이을 것인가}

### 특별 지시 (선택)
- 톤: {진지한/유머러스한/어두운/따뜻한 등}
- 강조할 부분: {전투/대화/감정/세계관 설명 등}
- 분량: {기본 6,000~7,000자, 조절 원하면 명시}
```

### 브레인스토밍 시 필요한 것

자유 형식으로 아래 중 **아무거나 1가지 이상** 제공:

- 관심 소재: "AI가 재판하는 세계", "조선시대 무협"
- 장르 선호: "현대 판타지 + 다크히어로"
- 레퍼런스: "나 혼자만 레벨업 같은 분위기인데, 좀 더 어둡게"
- 제외 요소: "하렘 없이", "학원물은 싫어"
- 타겟 독자: "20대 남성, 카카페 연재용"

## 3. 실전 사용 예시

### 예시 1: 신작 기획 (브레인스토밍)

```
💬 "SF 배경인데 마법도 있는 웹소설을 기획하고 싶어. 색다른 아이디어 줘"

🎭 Planner Layer 활성화 (브레인스토밍 모드)

📐 Plotter: 소재 카드 3장 생성
   ┣ 카드 1: "마법 운영체제" — 마법이 소프트웨어처럼 코딩되는 세계
   ┣ 카드 2: "마나 블록체인" — 마법 에너지가 분산 원장으로 관리
   └ 카드 3: "버그 헌터" — 세계의 물리법칙 버그를 사냥하는 직업

👤 "1번 카드가 재밌다. 이걸로 더 파줘"

🎭 Character + Stylist 합류
   세계관 5레이어 설계 + 인물 후보 카드 + 갈등 구조 + 독자 어필도 분석
```

### 예시 2: 연재 중 회차 집필

```
💬 "제20화 써줘. 한이 슬럼가에서 의문의 의뢰를 받는 회차야.
    전화 끝: 한이 전투에서 부상당해 슬럼가 은신처로 돌아옴
    핵심 사건: 어린 소녀가 의뢰를 가져오는데, 보수가 비정상적으로 높다
    클리프행어: 소녀의 정체가 상위 계층이라는 단서 발견"

🎭 PGE 전체 파이프라인 활성화

[Planner] 비트시트 + 인물노트 + 감각팔레트 + 대화설계
[Generator] 6,000~7,000자 원고 집필
[Evaluator] 5-PASS 검증 → 🟢 PASS 판정
```

### 예시 3: 부분 보강 (단일 역할 호출)

```
💬 "아래 전투 장면의 분위기가 밋밋해. 감각 묘사를 강화해줘:
    한은 검을 휘둘렀다. 적이 쓰러졌다. 
    주변이 조용해지자 한은 숨을 돌렸다."

🎭 Stylist Agent 단독 활성화 → 5감각 레이어링 적용
```

### 예시 4: 방향 고민 (혼합 모드)

```
💬 "3부 전개가 늘어지는 느낌이야. 다음 회차 방향을 바꿔서 써보자"

Phase A: Planner(브레인스토밍) → 갈등 강화 방안 3가지 제시
Phase B: 사용자 승인 (전환 게이트)
Phase C: Generator → 선택된 방향으로 원고 작성
Phase D: Evaluator → 최종 검증
```

## 4. 역할별 호출 가이드

| 이렇게 말하면 | 활성화되는 역할 |
|-------------|---------------|
| "비트시트 짜줘" | Plotter |
| "이 캐릭터 대사가 어색해" | Character |
| "이 장면 분위기를 바꿔줘" | Stylist |
| "원고 검토해줘" | Reviewer + Editor |
| "새로운 소재 아이디어 줘" | Plotter (기획 모드) |
| "세계관을 만들어줘" | Plotter + Stylist |
| "매력적인 악역 만들어줘" | Character |
| "갈등이 약해, 강화해줘" | Plotter |
| "이 아이디어 독자한테 먹힐까?" | Reviewer (시장성 검토) |

## 5. FAQ

### Q: 구상서가 없어도 쓸 수 있나요?
**네.** 줄거리를 직접 설명하면 됩니다. 다만 구상서가 있으면 설정 일관성 검증과 인물 아크 추적이 정확해집니다.

### Q: 분량을 조절할 수 있나요?
**가능합니다.** 기본은 6,000~7,000자(Hard Floor 5,500자). 조절 원하면 명시하세요.

### Q: 기존에 쓴 원고를 교정만 하고 싶다면?
교정 전문은 **`novel-proofreading-agent`** 스킬이 더 적합합니다.

### Q: 장르 제한이 있나요?
**없습니다.** 장르별 톤과 관습은 `novel-style-guide`의 장르 톤 매트릭스를 자동 참조합니다.
