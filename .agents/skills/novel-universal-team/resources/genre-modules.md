# 🎮 장르 모듈 정의서 (Genre Modules)

> **버전:** 2.0  
> **소속:** `novel-universal-team`  
> **역할:** 범용 팀의 파이프라인에 플러그인으로 결합되어 장르 특화 디테일을 강화

---

## 1. 모듈 인터페이스 표준

모든 장르 모듈은 동일한 인터페이스를 따른다:

```
Input:  범용 팀 에이전트의 산출물 (비트시트/인물노트/감각팔레트/원고)
Hook:   파이프라인 내 삽입 지점 (Plotter 후/Character 후/Stylist 후/Reviewer 후)
Output: 장르 특화 보강안 (삽입 위치 + 내용)
Scope:  모듈은 자기 장르 영역의 디테일만 다룸. 플롯·인물 구조 자체는 변경 불가
```

---

## 2. Fantasy Module (판타지)

**적용 장르:** 하이판타지, 다크판타지, 이세계, 게임판타지

| 항목 | 내용 |
|------|------|
| **강화 영역** | 마법 시스템, 종족 설정, 세계관 물리법칙, 스킬/능력 체계 |
| **Hook Points** | Plotter 후 (구조 변형), Stylist 후 (세계관 묘사 강화) |

**핵심 체크리스트:**
- [ ] 마법/능력 체계에 **한계(Cost)와 대가(Price)**가 명시되어 있는가?
- [ ] "왜 아무도 이 방법을 시도하지 않았는가?"에 답이 준비되어 있는가?
- [ ] 종족/세력 간 권력 구도가 갈등과 연결되어 있는가?
- [ ] 세계관 고유 용어가 과도하지 않은가? (초출 시 문맥으로 의미 전달)

**마법 시스템 설계 프레임워크:**
```markdown
### 힘의 체계: {시스템명}
- **원리:** {어떻게 작동하는가}
- **한계:** {무엇을 할 수 없는가}
- **대가:** {사용 시 치르는 비용}
- **예외:** {규칙을 벗어나는 사례와 그 이유}
- **사회적 영향:** {이 힘이 사회 구조에 미치는 영향}
```

## 3. Modern Fantasy Module (현대판타지)

**적용 장르:** 어반판타지, 현대+초자연, 히든월드, 헌터물

| 항목 | 내용 |
|------|------|
| **강화 영역** | 현실과 초자연의 경계 설계, 은폐 시스템, 일상↔비일상 전환 |
| **Hook Points** | Plotter 후 (이중 세계 구조), Character 후 (이중 정체성) |

**핵심 체크리스트:**
- [ ] 초자연 요소가 현실 세계에 노출되지 않는 이유(은폐 시스템)가 설득력 있는가?
- [ ] 주인공의 일상과 비일상의 전환이 자연스러운가?
- [ ] 현실 세계의 제도(경찰, 정부, 미디어)가 초자연에 어떻게 대응하는가?
- [ ] 현대 기술과 초자연 능력의 상호작용이 설계되어 있는가?

**이중 세계 설계 템플릿:**
```markdown
### 표면 세계 (Surface)
- 배경: {현대 도시/시골 등}
- 일반인의 인식: {초자연을 모름/소문 수준/공공연한 비밀}

### 이면 세계 (Hidden)
- 구조: {별도 차원/겹쳐진 공간/특정 장소}
- 진입 조건: {누가 어떻게 진입하는가}
- 은폐 메커니즘: {왜 일반인은 모르는가}

### 경계면 (Interface)
- 균열 지점: {두 세계가 겹치는 장소/상황}
- 드라마 잠재력: {경계 노출 시 발생할 갈등}
```

## 4. SF Module (사이언스 픽션)

**적용 장르:** 하드SF, 스페이스오페라, 사이버펑크, 포스트아포칼립스

| 항목 | 내용 |
|------|------|
| **강화 영역** | 과학적 설정의 내적 일관성, 기술 디테일, 미래 사회 구조 |
| **Hook Points** | Plotter 후 (기술 기반 갈등), Stylist 후 (테크노 묘사) |

**핵심 체크리스트:**
- [ ] 핵심 기술의 작동 원리가 (최소한 내적으로) 논리적인가?
- [ ] 기술이 사회·경제·정치에 미치는 파급 효과가 반영되어 있는가?
- [ ] 기술적 해결(Techno-fix)이 데우스 엑스 마키나가 되지 않는가?
- [ ] 전문 용어가 문맥 속에서 자연스럽게 설명되는가?

**기술 설정 카드:**
```markdown
### 기술: {기술명}
- **원리:** {기초 과학/가상 과학 설명}
- **현재 발전 단계:** {실험적/상용화/보편화}
- **접근성:** {누가 이 기술을 사용할 수 있는가}
- **부작용/위험:** {기술의 어두운 면}
- **서사적 역할:** {이 기술이 갈등에 기여하는 방식}
```

## 5. Fusion Module (장르 혼합)

**적용 장르:** 장르 크로스오버, 실험적 혼합 (무협+SF, 판타지+스릴러 등)

| 항목 | 내용 |
|------|------|
| **강화 영역** | 장르 간 톤 충돌 방지, 규칙 통합, 독자 기대 관리 |
| **Hook Points** | 전 단계 (장르 균형 감시) |

**톤 블렌딩 매트릭스:**
```markdown
### 혼합 장르: {장르A} × {장르B}

| 요소 | 장르A 기여 | 장르B 기여 | 블렌딩 규칙 |
|------|-----------|-----------|------------|
| 세계관 | {A의 세계관 요소} | {B의 세계관 요소} | {통합 방식} |
| 갈등 | {A의 갈등 유형} | {B의 갈등 유형} | {우선순위} |
| 문체 | {A의 톤} | {B의 톤} | {비중 A:B} |
| 독자 기대 | {A 독자의 기대} | {B 독자의 기대} | {양립 방안} |

### 충돌 방지 규칙
1. {규칙1}
2. {규칙2}
```

## 6. Romance Module (로맨스/감성)

**적용 장르:** 로맨스, 로맨스판타지, 감성 드라마, 힐링물

| 항목 | 내용 |
|------|------|
| **강화 영역** | 감정선 설계, 관계 역학, 로맨스 비트 배치 |
| **Hook Points** | Character 후 (관계 심화), Stylist 후 (감정 묘사 강화) |

**핵심 체크리스트:**
- [ ] 두 인물의 끌림에 **설득력 있는 이유**가 있는가?
- [ ] 관계 발전에 **장애물**(내적/외적)이 존재하는가?
- [ ] 감정 진행이 급발진 없이 **단계적**으로 고조되는가?
- [ ] 감정 묘사가 직서(Tell)가 아닌 간접(Show)으로 표현되는가?

**관계 아크 템플릿:**
```markdown
### {인물A} ↔ {인물B} 관계 아크
1. **첫인상:** {긍정적/부정적/무관심}
2. **접점 형성:** {두 사람이 가까워지는 계기}
3. **긴장 고조:** {끌림 + 장애물의 충돌}
4. **핵심 전환점:** {관계의 질이 바뀌는 결정적 순간}
5. **위기:** {관계가 깨질 수 있는 최대 위기}
6. **해소:** {위기를 통한 관계의 재정의}
```

## 7. Thriller Module (스릴러/서스펜스)

**적용 장르:** 법정스릴러, 미스터리, 서스펜스, 느와르, 범죄

| 항목 | 내용 |
|------|------|
| **강화 영역** | 긴장감 설계, 정보 비대칭, 시간 압박, 반전 배치 |
| **Hook Points** | Plotter 후 (서스펜스 구조), Stylist 후 (긴장감 묘사) |

**긴장감 설계 도구:**
```markdown
### 서스펜스 설계: 제{N}화

#### 정보 비대칭 맵
| 정보 | 독자 | 주인공 | 적대자 | 서스펜스 유형 |
|------|------|--------|--------|-------------|
| {정보A} | 알고 있음 | 모름 | 알고 있음 | 드라마틱 아이러니 |
| {정보B} | 모름 | 모름 | 알고 있음 | 미스터리 |
| {정보C} | 모름 | 알고 있음 | 모름 | 독자 호기심 |

#### 시간 압박
- 데드라인: {무엇이 언제까지 해결되어야 하는가}
- 카운트다운 방식: {명시적 시계/암묵적 진행/외부 사건}

#### 반전 배치
- 반전 위치: {회차 내 어디에서 반전}
- 복선: {이전에 심어둔 단서 목록}
- 페어플레이: {독자가 추리할 수 있는 단서 제공 여부}
```

## 8. Wuxia Module (무협/선협)

**적용 장르:** 정통 무협, 신무협, 퓨전 무협, 선협

| 항목 | 내용 |
|------|------|
| **강화 영역** | 무공 체계(심기체), 강호 세력 역학(구파일방 등), 은원 관계, 기연 |
| **Hook Points** | Plotter 후 (기연/복수 구조), Stylist 후 (무공 액션 묘사) |

**핵심 체크리스트:**
- [ ] 무공의 경지 상승이 단순한 수치 증가가 아닌 심상(心象)의 깨달음과 연결되는가?
- [ ] 인물의 행동 동기가 강호의 절대적 가치인 '은원(은혜와 원한)'에 기반하는가?
- [ ] 기연(비급, 영약)을 얻는 과정에 납득할 만한 시련과 대가가 존재하는가?

**무공/세력 설계 템플릿:**
```markdown
### 무공: {무공명}
- **특징:** {양강/음유/독공 등}
- **성취 단계:** {초식 → 검기 → 검강 → 심검 등}
- **리스크:** {주화입마의 조건 및 부작용}
```

## 9. Alt-History Module (대체역사/영지물)

**적용 장르:** 대체역사, 가상역사, 영지 발전물

| 항목 | 내용 |
|------|------|
| **강화 영역** | 역사적 분기점(나비효과), 기술/문화 발전 트리, 실존 인물 재해석 |
| **Hook Points** | Plotter 후 (역사적 사건 개입), Character 후 (시대적 가치관 충돌) |

**핵심 체크리스트:**
- [ ] 주인공의 개입으로 인한 역사적 나비효과가 논리적이고 입체적인가?
- [ ] 현대 지식(기술/제도)을 과거에 도입할 때, 당시의 시대적 한계와 마찰을 겪는가?
- [ ] 실존 인물들의 성격과 행동이 단순한 악역/조력자가 아닌 입체적으로 묘사되는가?

## 10. Profession Module (전문가물/스포츠)

**적용 장르:** 전문가물(의학, 요리, 재벌 등), 스포츠, 연예계

| 항목 | 내용 |
|------|------|
| **강화 영역** | 전문 지식의 디테일, 라이벌 구도, 업계 룰에 기반한 긴장감, 카타르시스 |
| **Hook Points** | Plotter 후 (성공/위기 플롯), Stylist 후 (전문 직무 묘사) |

**핵심 체크리스트:**
- [ ] 주인공의 전문적 성취가 독자에게 명확한 보상감(카타르시스)을 주는가?
- [ ] 피를 흘리는 전투 없이도, 업계의 규칙과 제약을 통해 텐션을 유발하는가?
- [ ] 라이벌이나 조력자가 주인공의 재능을 부각시키는 장치로 잘 활용되는가?

## 11. Horror Module (호러/괴담)

**적용 장르:** 코즈믹 호러, 규칙 기반 괴담, 서바이벌 호러

| 항목 | 내용 |
|------|------|
| **강화 영역** | 미지의 공포, 이성(Sanity) 붕괴, 불합리한 수칙 설계, 폐소공포증 |
| **Hook Points** | Plotter 후 (탈출/생존 플롯), Stylist 후 (기괴함/불안감 묘사) |

**핵심 체크리스트:**
- [ ] 괴이(Monster)의 정체나 규칙이 완전히 해명되지 않아 미지의 공포를 유지하는가?
- [ ] 주인공이 느끼는 무력함과 멘탈 붕괴의 과정이 감각적으로 묘사되는가?
- [ ] 규칙(수칙서)이 존재할 경우, 그 규칙을 어겼을 때의 페널티가 압도적인가?

## 12. 모듈 호출 프로토콜

```markdown
## 장르 모듈 호출 형식

### 호출
- **모듈:** {모듈명}
- **Hook:** {삽입 지점}
- **입력:** {범용 팀 산출물 요약}
- **요청:** {강화 요청 사항}

### 응답
- **보강 내용:** {구체적 추가/수정 내용}
- **삽입 위치:** {원고 내 삽입 위치}
- **근거:** {왜 이 보강이 필요한가}
```
