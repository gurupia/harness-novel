# 📏 유료 연재 분량 확보 가이드 (Volume Assurance Guide)

> **버전:** 1.0  
> **최종 수정:** 2026-04-23  
> **참조 위치:** `novel-writing-agent` §2.3~§2.5  
> **목적:** 유료 연재 플랫폼의 분량 기준(5,500자 이상)을 '물타기' 없이 밀도 있게 충족하기 위한 실전 참조 문서

---

## 1. 분량 기준 정의

### 1.1 플랫폼별 분량 기준

| 플랫폼 유형 | 최소 분량 | 목표 분량 | 비고 |
|------------|----------|----------|------|
| **카카오페이지/네이버시리즈** | 5,000자 | 6,000~7,000자 | 독자 만족도 최적 구간 |
| **문피아/노벨피아** | 4,500자 | 5,500~6,500자 | 장르에 따라 차이 |
| **에이전트 표준** | **5,500자** | **6,000~7,000자** | 모든 플랫폼 공통 최소선 |

### 1.2 분량 계산 규칙

- **포함:** 본문 텍스트, 대화문, 시스템 메시지, 의성어, 빈 줄(공백)
- **제외:** 제목(`# 제N화:`), 작가 노트, 메타데이터
- **도구:** 글자 수 측정은 '공백 포함' 기준을 사용한다

## 2. 분량 부족의 7가지 원인과 해결법

### 2.1 원인 진단표

| 원인 | 증상 | 해결 기법 |
|------|------|----------|
| **감각 묘사 부족** | 장면이 '스크립트'처럼 건조함 | 기법 1: 감각 레이어링 |
| **내면 묘사 부족** | 사건만 나열되어 감정 공백 발생 | 기법 2: 내면 교차 삽입 |
| **대화가 단순 전달** | 대사만 있고 인물의 살결이 없음 | 기법 3: 대화 확장 |
| **전투가 1~2합** | 적을 만나자마자 한 번에 처리 | 기법 4: 합(合) 확장 |
| **배경이 텅 빔** | 인물만 존재하는 진공 상태 | 기법 5: 환경 묘사 |
| **장면 전환이 급작스러움** | 독자가 장면 이동을 따라가지 못함 | 기법 6: 브릿지 삽입 |
| **단선적 서사** | 메인 플롯만 진행, 서사 깊이 부족 | 기법 7: 서브플롯 |

## 3. 기법별 상세 가이드

### 3.1 감각 레이어링 (Sensory Layering)

**원칙:** 핵심 장면에서 최소 3감각(시각·청각·촉각) 이상을 교차 배치한다.

**레이어링 체크리스트:**

| 감각 | 질문 | 확장 예시 |
|------|------|----------|
| 시각 | 무엇이 보이는가? (빛, 색, 거리, 움직임) | "안개 사이로 붉은 안광이 세 개, 삼각형을 이루며 떠올랐다." |
| 청각 | 무엇이 들리는가? (소음, 정적, 대비) | "금속이 긁히는 소리가 멈추자 자신의 심장 소리가 들렸다." |
| 촉각 | 피부에 무엇이 느껴지는가? (온도, 질감, 압력, 통증) | "비바람이 볼을 할퀴고 지나가며 열기를 빼앗았다." |
| 후각 | 무슨 냄새가 나는가? | "타버린 케이블에서 달콤하고도 독한 연기가 피어올랐다." |
| 미각 | 입안에 무엇이 느껴지는가? | "비릿한 금속 맛이 혀끝에 맴돌았다." |

**적용 빈도:**
- 새로운 공간 진입: **5감각 풀레이어링**
- 일반 장면: **3감각 이상**
- 액션 씬: **시각·촉각·청각** 필수

### 3.2 인물 내면 교차 삽입 (Internal Monologue Interweaving)

**원칙:** 사건 3~4문장마다 내면 1~2문장을 삽입하여 '감정의 맥박'을 유지한다.

**내면 묘사 유형:**

| 유형 | 설명 | 예시 |
|------|------|------|
| **즉각 판단** | 상황에 대한 순간적인 분석 | "놈의 공격 패턴은 직선적이다. 3번째에 반드시 빈틈이 온다." |
| **감정 반응** | 사건에 대한 감정적 반응 | "가슴 깊은 곳에서 차가운 공포가 올라왔다." |
| **과거 회상** | 유사한 과거 경험 연결 (1~2문장 이내) | "3년 전 공장에서 팔을 잃었을 때와 같은 소리였다." |
| **의지/결단** | 앞으로의 행동 의지 | "여기서 쓰러지면 모든 게 끝이다." |
| **자기 비꼼** | 캐릭터성을 드러내는 유머/자조 | "첫날부터 과로사라니, 운도 지지리 없군." |

**금지:**
- 내면 묘사가 3문장 이상 연속되는 것 (서사 흐름 정체)
- 이미 보여준 감정을 내면으로 재설명하는 것 (중복)

### 3.3 대화 확장

**공식:** `행동 묘사(1문장) → 대사 → 상대 반응(1문장) → 서브텍스트 행동`

```markdown
❌ 분량 부족형 대화 (30자):
"뭘 원하는 거야?"
"네 목숨."

✅ 풍성한 대화 (120자):
시우는 주먹을 꽉 쥐었다. 이클립스 핸드의 관절에서 금속음이 새어나왔다.
"뭘 원하는 거야?"
박 노인이 곰방대의 재를 떨었다. 연기 너머로 그의 기계 의안이 붉게 점멸했다.
"자네의 목숨이 아니라…… 자네의 선택이지."
```

### 3.4 전투 합(合) 확장

**최소 3합 규칙:**

각 합(合)은 다음 3요소를 포함한다:
1. **동작** (공격/방어/회피)
2. **결과** (성공/실패/부분 성공)  
3. **파급** (신체 비용/환경 변화/심리 변화)

```markdown
✅ 3합 전투 예시:

[1합: 선제]
시우가 이클립스 핸드를 휘둘렀다. 충격파가 드론의 장갑을 때렸지만,
겨우 도장이 벗겨진 수준이었다. 반동이 팔을 타고 올라와 어깨가 욱신거렸다.

[2합: 반격]  
드론의 집게가 시우의 옆구리를 할퀴고 지나갔다. 패딩이 찢기며
차가운 공기가 상처를 물었다. 시우는 이를 악물고 거리를 벌렸다.

[3합: 결착]
8호기의 보조 데이터가 약점을 특정했다. 관절 사이의 냉각 파이프.
시우는 전력으로 달려들며 정확히 그 지점에 주먹을 꽂았다.
파이프가 터지며 끓는 냉각수가 사방으로 튀었다.
```

### 3.5 환경 묘사의 서사적 활용

**원칙:** 배경은 '무대 장치'가 아니라 '등장인물'이다.

| 기법 | 예시 |
|------|------|
| **감정 반영** | "비가 점점 세차게 내렸다. 마치 하늘마저 시우의 절망에 동조하는 것처럼." |
| **설정 노출** | "산성비에 침식된 건물 외벽에는 에덴의 로고가 반쯤 녹아 있었다." |
| **시간 경과** | "달이 고철 산 너머로 기울기 시작했다. 새벽이 머지않았다." |
| **전투 참여** | "시우는 발밑의 고철 더미를 걷어차며 드론의 센서를 교란했다." |

### 3.6 브릿지 문단

**적용 시점:** 장면 전환이 일어나는 모든 곳.

```markdown
✅ 브릿지 예시:
시우는 타오르는 잔해를 뒤로 하고 지하 샤프트로 뛰어들었다.
낙하하는 동안 상처 난 옆구리에서 뜨거운 통증이 번져 나왔다.
— 살아 있다는 증거가 이렇게 아플 줄이야.

[장면 전환: 지하 공간]
어둠 속으로 추락한 그가 눈을 떴을 때, 전혀 다른 세계가 펼쳐졌다.
```

### 3.7 서브플롯 인터컷

**삽입 위치:** 메인 플롯의 전환점 직후 또는 클라이맥스 직전.
**분량:** 200~400자 (너무 길면 메인 플롯의 긴장감을 해침)

```markdown
✅ 적대자 인터컷 예시 (300자):
같은 시각, 화이트 타워 최상층.
카인 엘리자베스는 모니터에 떠오른 신호 데이터를 응시했다.
프로토타입 XB-01. 15년 전 그녀가 직접 폐기를 명령한 장비였다.
"살아 있었군."
그녀의 입꼬리가 올라갔다. 그것은 미소가 아니라 사냥감을 발견한
포식자의 반사 행동이었다.
"가디언 대대를 출동시켜. 생포가 아니라 회수다."
```

## 4. 분량 부족 시 긴급 보강 프로세스

분량이 5,500자 미만일 때 다음 순서로 보강한다:

```
1단계: 감각 레이어링 점검
   → 핵심 장면에 3감각 이상이 적용되었는가? → 미적용 시 추가 (+200~400자)

2단계: 내면 묘사 점검
   → 사건 사이에 내면이 5회 이상 삽입되었는가? → 미적용 시 추가 (+300~500자)

3단계: 대화 확장 점검
   → 핵심 대화에 행동+반응이 포함되었는가? → 미적용 시 추가 (+200~400자)

4단계: 전투 합 확장 (전투 회차)
   → 전투가 3합 이상인가? → 미달 시 합 추가 (+400~800자)

5단계: 서브플롯 추가
   → 서브플롯 장면이 1개 이상 있는가? → 없으면 인터컷 추가 (+200~400자)
```

## 5. 분량 과다 시 감축 가이드

분량이 8,000자를 초과할 때:

1. **중복 묘사 제거:** 동일 감정/상황을 2번 이상 서술한 곳을 찾아 1회로 통합
2. **설정 덩어리 압축:** 세계관 설명이 3문장 이상 연속되면 사건 속으로 분산
3. **회차 분할 검토:** 클라이맥스가 2개 이상이면 다음 화로 분리
4. **감각 과잉 조정:** 5감각 풀레이어링이 1회차에 3곳 이상이면 2곳으로 축소
