# 🔊 의성어·의태어 표준 DB (Sound Effects Guide)

> 장면 유형별 추천 의성어·의태어와 사용 한도를 정의한다.  
> 의성어는 **양념**이지 **주재료**가 아니다. 핵심 동작은 묘사로 전달한다.

---

## 사용 한도 규칙

| 규칙 | 기준 |
|------|------|
| 한 문단당 의성어 | **최대 2개** |
| 한 장면(연속 전투 등)당 의성어 | **최대 5~6개** |
| 같은 의성어 반복 | 한 회차 내 **동일 의성어 3회 초과 금지** |
| 장음 표기 | 같은 모음 반복 **최대 4회** (예: "콰아앙" ○, "콰아아아아앙" ✕) |
| 느낌표 | 연속 **최대 3개** |
| 의성어 단독 행 | 허용하되 **연속 2행 이상 의성어만으로 채우지 않음** |

## 전투·액션 장면

### 타격·충돌

| 의성어 | 용도 | 강도 |
|--------|------|------|
| 퍽, 턱 | 주먹 타격 (가벼운) | ★☆☆ |
| 쿵, 탕 | 중간 충돌, 바닥에 쓰러짐 | ★★☆ |
| 쾅, 콰앙 | 강한 충돌, 벽 파괴 | ★★★ |
| 콰광, 쾅- | 대규모 폭발, 건물 붕괴 | ★★★ |

### 검·무기

| 의성어 | 용도 |
|--------|------|
| 챙, 챙그랑 | 금속 교차, 검의 충돌 |
| 스윽, 서걱 | 검을 빼는 소리. 날카로운 일격 |
| 슈웅, 쉬이잉 | 빠른 베기, 공기를 가르는 소리 |
| 쩍, 쩌억 | 갑옷/방패 파쇄 |
| 푹, 철퍽 | 관통, 찌르기 |

### 마법·에너지

| 의성어 | 용도 |
|--------|------|
| 웅, 위이잉 | 마력 충전, 기 집중 |
| 파앙, 파직 | 마법 발사, 에너지 방출 |
| 치직, 파지직 | 전기/스파크 계열 |
| 훠어어 | 화염/열기 확산 |
| 콰르륵 | 대지/바위 계열 마법 |

## 감정·분위기 장면

| 의태어/의성어 | 용도 |
|-------------|------|
| 덜덜, 파들파들 | 두려움으로 인한 떨림 |
| 흐느적, 비틀비틀 | 기진맥진한 움직임 |
| 와락 | 갑작스러운 포옹, 움켜쥠 |
| 후두둑 | 눈물이 떨어지는 소리 |
| 스윽 | 조용한 미소, 눈길을 돌림 |
| 꿀꺽 | 긴장감의 표현(침을 삼킴) |

## 환경·자연 장면

| 의성어 | 용도 |
|--------|------|
| 우르르, 쿠구구 | 천둥, 지진, 큰 진동 |

## 의성어 대체 전략

의성어 사용 한도를 초과한 경우, **묘사적 표현으로 대체**한다.

| 의성어 원본 | 묘사적 대체 |
|-----------|------------|
| "콰앙!" | "충격파가 갈비뼈를 울렸다" |
| "챙그랑!" | "검날이 부딪히며 불꽃이 튀었다" |
| "삐이익!" | "경보음이 고막을 찢었다" |
| "위이잉" | "모터가 저주파 진동을 토해냈다" |
