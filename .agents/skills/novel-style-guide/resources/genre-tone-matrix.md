# 🎭 장르별 톤 매트릭스 (Genre Tone Matrix)

> 각 장르의 **어조, 시점, 문장 리듬, 비유 관습, 금기**를 정의한다.  
> 집필 시 장르의 톤앤매너를 유지하고, 교정 시 이탈 여부를 판단하는 기준으로 사용한다.

---

## 무협 (武俠)

| 항목 | 기준 |
|------|------|
| **시점** | 3인칭 제한 시점 (주인공 중심). 전지적 시점은 세력전 장면에서만 제한적 허용 |
| **서술 톤** | 무겁고 고풍스러운 한문체 혼용. 비장미와 호쾌함의 교차 |
| **문장 리듬** | 전투: 짧고 날카로운 단문. 수련/일상: 중문 혼합. 내공 묘사: 서정적 장문 |
| **비유 관습** | 자연물(산, 바람, 물, 달), 동물(용, 학, 호랑이), 무기/검법의 의인화 |
| **전문 어휘** | 내공, 진기, 경맥, 단전, 심법, 초식, 검기, 검강, 기연, 영약 |
| **금기** | 현대 비유("미사일처럼", "컴퓨터 같은"), 지나친 경어체, 외래어 남용 |

## 판타지 / 퓨전 판타지

| 항목 | 기준 |
|------|------|
| **시점** | 3인칭 제한 시점. 1인칭도 허용 (주인공의 내면 독백이 많은 경우) |
| **서술 톤** | 장엄하되 가독성 유지. 퓨전의 경우 현대적 유머 혼합 가능 |
| **문장 리듬** | 세계관 설명: 중문~장문. 전투/이벤트: 단문 위주. 대화: 캐릭터별 차별화 |
| **비유 관습** | 마법적 자연현상, 신화적 존재, 원소(불/물/바람/땅) 기반 비유 |
| **전문 어휘** | 서클/티어, 마나, 오라, 스킬, 마정석, 마탑, 던전, 길드 |
| **금기** | 과도한 게임 용어(아이템 박스, 가챠), 현실 브랜드명, 메타적 발언 |

## 현대 판타지 / 헌터물

| 항목 | 기준 |
|------|------|
| **시점** | 3인칭 제한 시점 또는 1인칭 |
| **서술 톤** | 현실적이고 건조한 기반 + 초자연 요소의 긴장감. "일상에 균열이 생긴" 느낌 |
| **문장 리듬** | 일상: 구어체에 가까운 단문. 던전/전투: 긴박한 단문 연속. 각성/레벨업: 감각적 묘사 |
| **비유 관습** | 현대 기술 + 초자연의 대비("네온사인 아래 검은 게이트가 열렸다") |
| **전문 어휘** | 각성, 헌터, 게이트, 던전, S급~F급, 레이드, 마석, 기르드, 협회 |
| **금기** | 중세 판타지 어투("여봐라", "그대"), 지나친 이세계 문법 혼용 |

## 사이버펑크 / SF

| 항목 | 기준 |
|------|------|
| **시점** | 3인칭 제한 시점 또는 1인칭 (하드보일드 내레이션) |
| **서술 톤** | 냉소적, 건조한, 기술적. 감정은 절제하되 행간에 숨김 |
| **문장 리듬** | 짧고 건조한 단문 위주. 시스템 메시지와 서술의 교차. 감정 폭발 시에만 장문 허용 |
| **비유 관습** | 디지털/공학 은유("메모리가 오버플로우된 것처럼", "시냅스가 단락되었다"), 기계/회로/네트워크 |
| **전문 어휘** | 코어, 시냅스, 해킹, 프로토콜, 노드, 인터페이스, 외골격, 뉴럴링크, 마나 네트워크 |
| **금기** | 목가적/전원적 비유("봄날의 들판처럼"), 중세 판타지 어투, 동화적 표현 |

## 게임 판타지

| 항목 | 기준 |
|------|------|
| **시점** | 3인칭 제한 시점 또는 1인칭 |
| **서술 톤** | 경쾌하고 리듬감 있는 서술. 상태창/시스템 메시지가 서사와 자연스럽게 융합 |
| **문장 리듬** | 시스템 메시지 삽입으로 인한 잦은 리듬 전환. 전투 = 빠른 호흡, 탐색 = 여유로운 호흡 |
| **비유 관습** | 게임 메커니즘 자체가 비유의 원천("버프가 걸린 것처럼 몸이 가벼웠다") |
| **전문 어휘** | 레벨, 스킬, 상태창, 인벤토리, 퀘스트, 경험치, 드롭, 보스, 레이드, 파티 |
| **금기** | 시스템 부정("이건 게임이 아니야"류의 과도한 메타 인식), 시스템 없이 무한 파워업 |

## 로맨스 / 로맨스 판타지

| 항목 | 기준 |
|------|------|
| **시점** | 1인칭 또는 3인칭 제한 시점 (여주 중심). 남주 시점 교차 가능 |
| **서술 톤** | 감성적이고 섬세한. 내면의 감정 변화를 정밀하게 묘사 |
| **문장 리듬** | 감정 고조: 짧은 문장 + 독백. 일상/설레임: 중문의 부드러운 리듬. 갈등: 날카로운 대화 |
| **비유 관습** | 꽃, 계절, 빛/그림자, 체온/향기 등 감각적 비유 |
| **전문 어휘** | 장르에 따라 다름 (궁중: 내전/황실. 아카데미: 학원/사교. 현대: 재벌/연예계) |
| **금기** | 과도한 전투 묘사, 잔혹/고어 장면, 주인공의 감정을 건너뛰는 요약적 서술 |

## 법정 / 수사 / 스릴러

| 항목 | 기준 |
|------|------|
| **시점** | 3인칭 제한 시점 또는 1인칭 (주인공 = 탐정/검사/형사) |
| **서술 톤** | 긴장감 있고 절제된. 감정보다 논리와 관찰을 우선 |
| **문장 리듬** | 추리/분석: 논리적 중문. 심문/재판: 날카로운 대화체. 추격/위기: 빠른 단문 |
| **비유 관습** | 체스/게임 비유("수를 놓다"), 시계/시간 비유("타이머가 돌아가고 있었다") |
| **전문 어휘** | 증거, 공소, 변론, 판결, 알리바이, 동기, 포렌식, 프로파일링 |
| **금기** | 비현실적 능력(초능력으로 범인 찾기), 법률 지식의 명백한 오류, 감정적 판결 |

## 성인물 (19금)

| 항목 | 기준 |
|------|------|
| **시점** | 1인칭 또는 3인칭 제한 시점 |
| **서술 톤** | 관능적이되 품위 유지. 포르노가 아닌 에로티시즘 지향. 감정선이 동반되어야 함 |
| **문장 리듬** | 감각 묘사: 느린 장문. 절정부: 짧고 호흡이 끊어지는 단문. 여운: 서정적 중문 |
| **비유 관습** | 자연현상(파도, 폭풍, 불꽃), 감각(체온, 향기, 촉감)을 통한 간접 표현 |
| **금기** | 의학 용어 직접 사용, 지나치게 기계적인 묘사, 동의 없는 행위의 미화 |

## 역사 / 대체역사

| 항목 | 기준 |
|------|------|
| **시점** | 3인칭 제한 시점. 전지적 시점은 대규모 역사적 장면에서만 허용 |
| **서술 톤** | 시대의 무게감을 살리되 가독성 유지. 존칭/경어 체계는 시대 설정에 맞춤 |
| **문장 리듬** | 궁정/정치: 격식 있는 장문. 전장: 박력 있는 단문. 일상: 시대의 정서를 반영한 중문 |
| **비유 관습** | 해당 시대에 존재했던 사물·현상만 사용. 현대 비유 절대 금지 |
| **금기** | 시대착오적 언어("OK", "스트레스"), 역사적 사실의 의도적 왜곡(대체역사 설정 제외) |
