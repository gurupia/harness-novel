# 💻 시스템 메시지 포맷 가이드 (System Message Templates)

> 게임판타지, 사이버펑크, 헌터물 등 **시스템 UI가 작품 내에 등장하는 장르**에서  
> 상태창, 경고 메시지, 퀘스트 알림 등을 작성할 때의 포맷 규칙을 정의한다.

---

## 공통 규칙

| 규칙 | 기준 |
|------|------|
| **길이** | 한 메시지당 **최대 80자(한글 기준)** 또는 **영문 키워드 5개 이내** |
| **형식** | 대괄호 `[ ]` 로 감싸거나, 별도 서식(들여쓰기, 기울임 등)으로 본문과 구분 |
| **언어** | 영문 키워드 + 한글 설명의 혼합 허용. 순한글 또는 순영문도 가능 |
| **빈도** | 한 회차(5,000자) 내 시스템 메시지 **최대 8~10개** |
| **배치** | 서사의 흐름을 끊지 않는 위치에 삽입. 액션 중간에 긴 메시지 삽입 금지 |
| **동의어 금지** | 시스템 메시지 내에서도 동의어 나열(§2 규칙) 적용 |

---

## 장르별 포맷 템플릿

### 사이버펑크 / 디지털 세계관

```markdown
[Warning: Core Temperature — 92% DANGER]
[Status: Neural Sync Active / Shield Integrity 45%]
[FATAL ERROR: Central Logic Crash — Shutdown Imminent]
[System: 'All-Clear' Protocol Activated]
[Destination Locked: The Sky Garden]
```

**톤:** 기계적이고 건조한 영문 기반. 감정 배제. 명사+수치 위주.
**금지:** 시스템이 감정적 표현을 사용하는 것 ("아아! 파괴되고 있습니다!")

### 게임 판타지 / 상태창 세계관

```markdown
[레벨 업! Lv.34 → Lv.35]
[새로운 스킬 획득: 〈그림자 도약〉 (A급)]
[퀘스트 완료! 보상: 경험치 50,000 / 골드 3,000]
[경고: HP가 15% 미만입니다. 포션 사용을 권장합니다.]
[숨겨진 던전 발견: 〈잊혀진 왕의 무덤〉]
```

**톤:** 게임 인터페이스를 충실히 재현. 수치와 등급이 명확.
**금지:** 설명이 지나치게 길거나 소설적 문장("`당신은 위대한 전사로 거듭났습니다...`")

### 현대 판타지 / 헌터물

```markdown
[각성 감지! 마력 수치: 12,400 (S급 기준 초과)]
[게이트 등급: A+ / 예상 마수 출현: 4종]
[긴급 알림: 서울 강남 B구역 — 레드 게이트 발생]
[레이드 참가 자격 확인 — 승인 완료]
```

**톤:** 현실의 공공 알림 시스템 + 판타지 수치. 뉴스 속보 같은 느낌.
**금지:** 과도한 판타지 어투("`신의 축복이 내려졌도다`")

### 무협 (시스템 없는 장르)

무협에서는 일반적으로 시스템 메시지를 사용하지 않는다.  
**회귀 무협** 등 시스템이 존재하는 경우에만 아래 형식을 허용:

```markdown
[심법 수련 진행: 태극심결 3성 → 4성]
[경고: 주화입마 징후 감지. 즉시 운기조식 필요]
[기연 감지: 북방 설산 동굴 — 빙한지옥에서 영약 반응]
```

**톤:** 한문체 혼용. 무협 세계관 용어를 시스템 언어로 치환.

## 시스템 메시지 안티 패턴

아래 패턴은 어떤 장르에서도 금지한다.

### ❌ 동의어 폭탄

```markdown
[Super Ultimate Critical Infinite Dead System Fatal Error 404: 
 거대 리밋 부서짐 오버플로우 한계 오류 폭주 파괴! 
 신성 중앙 보드 연산 대뇌 크래시 삭제 이탈! 다운! 통제 박살 권한 삭제 사망! 셧 다운!!!]
```

**수정:** `[FATAL ERROR: Core Crash — System Shutdown]`

### ❌ 소설적 장황함

```markdown
[Network Protocol Infrastructure Current_Status: 
 권력 마나 제국 중앙 집권 사유 독재 사망 포맷 소실 삭제 -> 
 새로운 파워 거점 코어. 글로벌 분산 엣지 대중 수백만 빈민 슬럼 직접 병렬 연결 
 평등 무제한 투하 전송 오픈 살포 배포 완료 도래 성공! 해방 체제 독립 완성!]
```

**수정:** `[Status: Central Authority — TERMINATED / Distributed Access — ACTIVE]`

### ❌ 시스템의 감정 표현

```markdown
[시스템이 기뻐합니다! 당신은 정말 대단한 영웅입니다!!! 
 이 세상을 구한 당신에게 무한한 축복을 내립니다~~~]
```

**수정:** `[Quest Complete. Reward Distributed.]` (시스템은 감정이 없다)
